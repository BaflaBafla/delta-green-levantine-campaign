# Assets Manifest

This folder contains table assets, images, and handouts to support The Levantine Engine campaign. Use these files as printable or virtual-table assets. If you want alternate formats or higher-resolution images, tell me which asset and target format (PNG/SVG/PDF) and I will produce them.

Files included:

- google_briefing.svg — Four-tab briefing image (A4 landscape SVG) for Session 1 handout.
- binding_mark.svg — Stylized Binding Mark (hexagram + fish) as an SVG for print/props.
- session_zero.md — Session Zero handout: content warnings, lines & veils, consent form, debrief plan.
- mechanical_faq.md — Compact FAQ covering Pact / Static timing, WP economy, Group Bond mechanics, and examples.
- player_reference_card.md — One-page quick reference (text) for players: Pact, Static, Group Bond summaries.
- ossuary_photo_instructions.md — Step-by-step instructions to produce an ossuary photo/prop and recommended image sizes.
- manifest.md — This manifest file (same content as Assets Manifest header)
- readme.md — Short usage guide for the Assets folder

If you'd like, I can also generate PNG exports of the SVGs at print resolution (A4 300 dpi) and add them to the folder, if you want.
