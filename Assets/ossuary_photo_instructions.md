# Ossuary Photo — Production Notes

Purpose: a realistic-looking ossuary photo/print to hand players when they enter the Nablus cave. The image should show a limestone ossuary with a recarved rosette replaced by the Binding Mark.

Quick method (low-cost):
1. Find a public-domain photo of a 1st-century Judean ossuary (Wikimedia Commons). Prefer a rectangular limestone box image.
2. Open in an image editor (GIMP, Photoshop). Add a new layer and overlay the Binding Mark (use Assets/binding_mark.svg as a template). Blend mode: multiply/overlay; reduce opacity to 85%.
3. Add subtle ageing: noise + slight sepia + vignette. Add a small scratch/soil texture at the corner. Save as 4x6 inch at 300dpi (1200×1800 px).
4. Print on matte photo paper. Optional: brush a thin layer of diluted PVA glue with a drop of brown paint to simulate dirt (use sparingly).

Digital VTT use:
- Provide a 2048×1536 PNG export (maintain aspect ratio) and a 512×384 thumbnail for quick lookup.

Legal: ensure source image is public domain or properly licensed for modification. If unsure, request that I create a custom, fictional ossuary graphic instead.
