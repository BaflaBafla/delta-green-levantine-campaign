# Assets README

This folder holds assets intended for immediate use at the table and for virtual tabletops. Files:

- google_briefing.svg — printable A4 landscape 4-tab briefing. Edit text layers in an SVG-aware editor.
- binding_mark.svg — vector art for the Binding Mark (hexagram + fish). Use for props, laser-cut stencils, or layered images.
- session_zero.md — Session Zero handout (content warnings, lines/veils, safety tools).
- mechanical_faq.md — Rules clarifications and examples.
- player_reference_card.md — Quick cheat for players.
- ossuary_photo_instructions.md — How to make an ossuary photo prop.
- manifest.md — asset manifest listing all files.

If you want PDF exports, PNG renders, or alternate aspect ratios, tell me which assets and target formats; I will produce them and add them to this folder.
